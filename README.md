# Movie Recommendation & Rating System

## Overview

A Python and SQLite based application that allows users to manage movies, provide ratings and reviews, filter movies by genre, and receive personalized movie recommendations.

## Features

* Add Genres
* Add Movies
* Register Users
* Rate Movies
* Write Reviews
* View Top Rated Movies
* Filter Movies by Genre
* Personalized Movie Recommendations

## Technologies Used

* Python
* SQLite
* SQL
* Git
* GitHub

## Database Tables

* genres
* movies
* users
* ratings

## SQL Concepts Used

* JOIN
* AVG()
* GROUP BY
* ORDER BY
* Filtering

## How to Run

1. Run database.py
2. Run main.py
3. Use the menu-driven interface
